Instant Compose

Start building your Compose app within a minute using any editor.

Get Started Learn about Compose GitHub

Get Started

1 Install the CLI 
curl -fsSL https://emilflach.github.io/instant-compose/get.sh | bash
2 Create your app 
instant-compose init myApp
cd myApp
./gradlew :dev:run
Troubleshooting
A Missing curl? 
apt-get update && apt-get install curl
B Refresh Terminal 
source ~/.bashrc
or 
source ~/.zshrc
